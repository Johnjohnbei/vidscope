# S02: WatchRefreshUseCase + vidscope watch CLI sub-application

**Goal:** Wire the watchlist CRUD + refresh logic end-to-end via the CLI. Ship WatchRefreshUseCase that iterates watched accounts, lists recent videos via list_channel_videos, dedupes against existing videos, runs new ones through the existing pipeline_runner. Ship `vidscope watch add/list/remove/refresh` CLI commands.
**Demo:** After this: `vidscope watch add https://www.youtube.com/@channel` registers the account. `vidscope watch list` shows the table. `vidscope watch refresh` iterates accounts, lists recent videos per account, deduplicates against existing videos, runs new ones through the pipeline. Idempotent (second run = 0 new).
