# S01: Project socle, data layer and CLI skeleton

**Goal:** Build the VidScope socle: installable package via uv, data layer with the full SQLite schema (videos, transcripts, frames, analyses, pipeline_runs, FTS5), a Typer CLI skeleton with all five planned subcommands as working stubs, a config module resolving data/cache/db paths, startup checks for ffmpeg and yt-dlp, and a pytest+ruff+mypy baseline that future slices can rely on.
**Demo:** After this: a fresh clone installs via `uv tool install .`, `vidscope --help` lists all planned subcommands as stubs, the SQLite DB is created with the full schema on first run, and `vidscope status` returns an empty-but-valid report.
