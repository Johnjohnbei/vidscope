# S03: Transcription brick (faster-whisper)

**Goal:** Add transcription to the pipeline: a FasterWhisperTranscriber adapter behind the existing Transcriber port, a TranscribeStage that orchestrates Transcriber + MediaStorage + TranscriptRepository, container wiring, and integration tests proving real transcription works on real downloaded media. The default model is faster-whisper's `base` model (CPU-friendly, multilingual, ~150MB cached locally on first use). French and English are both verified.
**Demo:** After this: `vidscope add <url>` on a public Reel produces a full transcript and timestamped segments stored in the DB, with wall-clock timing recorded in the pipeline run; French and English both verified.
